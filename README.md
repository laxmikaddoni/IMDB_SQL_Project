## Project Title
IMDB movies & directors Database Analysis Using SQL

## Project Overview
This project focuses on analyzing an IMDB-style relational database using SQL.
The objective is to retrieve, analyze, and interpret movie and director data to understand trends related to popularity, revenue, ratings, and director performance.
The project helps in gaining practical experience in database querying and data analysis.

## Tools Used
SQL – For querying and data analysis
MySQL – Database management
MySQL Workbench – Query execution

## Database Details
directors Table
id – Director unique ID
name – Director name
gender – Gender (0/2 = Male, 1 = Female)
department – Department (Directing)

## movies Table
id – Movie unique ID
title – Movie title
popularity – Popularity score
revenue – Movie revenue
vote_average – Average rating
vote_count – Total votes
release_date – Release date
director_id – Foreign key referencing Directors table

## Tasks Performed
Retrieved complete movie and director data
Counted total movies in the database
Filtered directors by name and alphabet
Analyzed gender distribution among directors
Identified top popular and bankable movies
Found highest rated movies after the year 2000
Determined directors with most movies and highest revenue

## Key Insights
Popular movies are not always the highest revenue earners
A few directors dominate movie production
Female directors are underrepresented in the dataset
Movies released after 2000 generally have higher ratings
Director reputation strongly influences movie success

## Conclusion
This project successfully demonstrates the use of SQL for analyzing structured movie data.
It provides valuable insights into movie performance and director impact while strengthening skills in SQL, data analysis, and database management.
The project serves as a strong foundation for future work in data analytics and business intelligence.
